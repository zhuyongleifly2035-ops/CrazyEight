# 🃏 Crazy Eights (疯狂 8 点)

一个基于 React + Tailwind CSS + Motion 构建的经典扑克牌游戏。支持 AI 对战、流畅动画和响应式布局。

## 🌟 游戏特性
- **经典规则**：完整的疯狂 8 点规则，包括万能 8 点换色功能。
- **智能 AI**：具有基础策略的 AI 对手。
- **丝滑动画**：使用 `motion` 实现的发牌、出牌和交互动画。
- **响应式设计**：完美适配手机、平板和电脑。
- **视觉效果**：精美的卡牌设计和胜利庆祝特效。

## 🛠 技术栈
- **前端框架**: React 19
- **样式**: Tailwind CSS 4
- **动画**: Motion (原 Framer Motion)
- **图标**: Lucide React
- **部署**: Vercel

## 🚀 本地开发
1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
3. 构建项目：
   ```bash
   npm run build
   ```

## 📦 部署到 Vercel
本项目已包含 `vercel.json` 配置文件，支持单页应用路由。直接连接 GitHub 仓库即可一键部署。
